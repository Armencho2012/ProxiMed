# ProxiMed: Diabetes Risk Prediction via Custom KNN

> ⚡ **100% Pure Python Engineering.**
> An end-to-end medical screening application that predicts diabetic risk using a custom-built K-Nearest Neighbors (KNN) spatial classification algorithm, interactive desktop UI, and manual evaluation metric engines trained on over 250,000 CDC health records.

---

## 1. Project Overview

Diabetes is a highly prevalent chronic condition globally. Because clinical diagnostics require invasive blood draws, ProxiMed offers a non-invasive screening alternative. It maps self-reported lifestyle choices, demographics, and self-assessments directly to diabetic risk categories in real time.

To demonstrate pure software engineering fundamentals, every piece of mathematical modeling—including feature scaling, spatial distance searching, nearest-neighbor voting, and confusion matrix evaluations—is built completely from scratch without external machine learning libraries.
